Descripción:
Este proyecto consiste en el desarrollo de una aplicación de tipo
"billetera virtual" programada en Java. 
Este proyecto consiste en el desarrollo de una aplicación de tipo
"billetera virtual" programada en Java, diseñada para brindar una 
gestión eficiente y accesible de finanzas personales. La aplicación 
incorpora tecnología moderna y emplea principios sólidos de desarrollo 
de software, como la utilización de patrones de diseño ampliamente 
recomendados en Java (por ejemplo, Singleton,....TODO ) para
hacerla modular, mantenible y escalable. 

La aplicación no solo se enfoca en ofrecer funcionalidad básica, sino 
también en garantizar una experiencia de usuario intuitiva, segura y 
personalizable. Además, el proyecto se plantea como una solución educativa 
que abarca desde la autenticación robusta hasta la generación automática 
de informes financieros, destacando así su carácter didáctico y formativo.

La estructura del proyecto incluye:

- Módulo de autenticación e inicio de sesión.
- Gestión de usuarios (registro, edición y eliminación).
- Manejo de categorias, presupuestos, saldos y transacciones.
- Implementación de medidas de seguridad (encriptación y validación).
- Interfaz de usuario básica para una experiencia intuitiva.
- Integración de almacenamiento local o en la nube.
- Utilización de patrones de diseño recomendados en Java.
- Sección de informes y estadísticas de uso.
